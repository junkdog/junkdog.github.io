takeWhile

fun <A> takeWhile(p: (A) -> Boolean): Transducer<A, A> (source)

Creates a transducer that transforms a reducing function such that it processes inputs as long as the provided predicate returns true. If the predicate returns false, the reducing process stops.

Parameters

p - a predicate used to test inputs

A - input type of input and output reducing functions

Return
a new transducer
