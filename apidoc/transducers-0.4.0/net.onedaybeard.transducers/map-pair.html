mapPair

inline fun <B, reified K : Any, reified V : Any> mapPair(l: Xf<K, B>, r: Xf<V, B>): Xf<Pair<K, V>, B> (source)

Returns a transducer creating Pairs, with values provided via transducers l and r. Each value transducer runs a full transducible process/lifecycle per input, allowing for more sophisticated transducers at the cost of performance.

listOf(mapPair(map { i: Int -> i % 2 },
               map { i: Int -> i * 2 }),
       (0..5)
) assertEquals listOf(
    0 to 0,
    1 to 2,
    0 to 4,
    1 to 6,
    0 to 8,
    1 to 10)
mapOfLists(xf = mapPair(l = map { i: Int -> i % 2 } +
                            map(Int::toString),
                        r = map { i: Int -> i * 2 }),
           input = (0..9)
) assertEquals mapOf("0" to listOf(0, 4, 8, 12, 16),
                     "1" to listOf(2, 6, 10, 14, 18))
mapOf(mapPair(l = map { i: Int -> i % 2 } +
                  map(Int::toString),
              r = map { i: Int -> i * 2 }),
      ::mutableSetOf,  // group into collection
      (0..9)
) assertEquals mapOf("0" to setOf(0, 4, 8, 12, 16),
                     "1" to setOf(2, 6, 10, 14, 18))
