subducing

inline fun <reified A : Any, B> subducing(xf: Xf<A, B>): Rf<A, B> (source)

Returns a specialized ReducingFunction for xf, treating each input as a full transducible process - invoking each of the three ReducingFunction.apply functions once.

An initial result is computed based on the type of A.
