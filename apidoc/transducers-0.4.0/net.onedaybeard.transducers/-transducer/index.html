interface Transducer<out B, C> (source)

A Transducer transforms a reducing function of one type into a reducing function of another (possibly the same) type, applying mapping, filtering, flattening, etc. logic as desired.

Parameters

B - The type of data processed by an input process

C - The type of data processed by the transduced process
