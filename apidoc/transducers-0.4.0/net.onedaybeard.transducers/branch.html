branch

fun <A, B> branch(test: (B) -> Boolean, xfTrue: Transducer<A, B>, xfFalse: Transducer<A, B>): Transducer<A, B> (source)

Returns a branching transducer, routing each input through either xfTrue or xfFalse, depending on the value returned by test.

intoList(xf = copy<Int>() +
              branch(test = { it % 4 == 0 },
                     xfTrue  = map { i: Int -> i * 100 },
                     xfFalse = map { i: Int -> i / 4 } +
                               distinct() +
                               map(Int::toString)),
         input = (1..9)
) assertEquals listOf("0", 400, 400, "1", 800, 800, "2")
